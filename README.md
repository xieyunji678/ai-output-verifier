# AI Output Verifier

一个用于验证 AI Agent 输出内容真实性的平台。

## 功能

- ✅ 事实核查 - 验证 AI 输出中的事实准确性
- ✅ 幻觉检测 - 识别 AI 胡编乱造的内容
- 📊 可视化标注 - 直观展示可疑内容
- 📁 历史记录 - 保存验证结果
